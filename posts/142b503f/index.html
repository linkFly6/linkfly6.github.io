结语

装饰器目前在 ECMAScript 新提案中的 建议征集的第二阶段(Stage 2)，由于装饰器在其他语言中早已实现，例如 Java 的注解(Annotation) 和 C# 的特性(Attribute)，所以纳入 ECMAScript 规范只是时间问题了。
